# University Full-Stack Web Application

This project is a full-stack web application developed as a university project.

## Technologies Used
- React
- JavaScript
- Tailwind CSS
- Python
- MongoDB

## Features
- Responsive user interface
- Frontend and backend integration
- REST API communication
- MongoDB database operations
- Admin login and dashboard
- Dynamic content management

## Project Structure
- frontend/: React-based frontend application
- backend/: Python-based backend API
- database/: MongoDB integration

## Purpose
The purpose of this project is to demonstrate full-stack web development skills, including frontend-backend communication, database integration, and modern UI design.

## Setup
1. Install frontend dependencies:
npm install
2. Start frontend:
npm start
3. Configure backend environment variables
4. Run backend server

## Notes
Environment variables and dependency folders are excluded from version control for security and best practices.
